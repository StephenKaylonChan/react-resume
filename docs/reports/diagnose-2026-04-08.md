### P0 — 违反项目规则（2 个）

**#1** `[D11]` `src/components/GradientText.jsx:51-57` `[置信度: 确定]`

GradientText 组件使用 `style={{ WebkitBackgroundClip: 'text', ... }}` 内联样式。
前端规则明确要求 "MUST NOT 使用 style={{}}，MUST 使用 Tailwind CSS utility class"。
| 依据：`.claude/rules/frontend.md` 第一条规则

**#2** `[D5/D11]` `src/components/GradientText.jsx:51-57` + `src/index.css:76-85` `[置信度: 确定]`
**渐变文字样式双重定义**

GradientText 组件又通过 `style={{}}` 重复设置完全相同的属性。逻辑重复且违反规则。
| 依据：对比 index.css:76-85 和 GradientText.jsx:51-57

### P2 — 应修复（8 个）

**#8** `[D6]` `src/utils/exportPDF.js:69-71` `[置信度: 确定]`
**img.onload 缺少错误处理，可能导致 Promise 永远 pending**
```javascript
await new Promise((resolve) => {
  img.onload = resolve;
});
```
未设置 `img.onerror`，若图片 dataUrl 异常，Promise 将永远不 resolve/reject，导致 UI 卡在"生成中…"状态。
| 依据：exportPDF.js 第 69-71 行

**#9** `[D8]` `src/components/ResumeHeader.jsx:9-37` `[置信度: 确定]`

`icons` 对象包含 5 个 SVG JSX 元素，定义在 ContactIcon 函数内部，每次组件渲染都会重新创建。应移到组件外部作为模块级常量。
| 依据：ResumeHeader.jsx ContactIcon 函数体

**#10** `[D8]` `src/components/GradientText.jsx:13-44` `[置信度: 确定]`
