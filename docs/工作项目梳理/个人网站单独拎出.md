## 项目 6：个人主页（kaylonchan-website）

### 1. 这个项目是干什么的（一句话）

集旅行摄影展示、技术博客、日语学习平台、开发者工具于一体的全栈个人品牌网站，采用 Monorepo 架构管理 3 个前端应用 + 1 个后端 API + 5 个共享包。

### 2. 我负责的具体模块（全栈独立设计 + 开发）

**前端主站（website-web）**：

- 旅行摄影展示、博客系统、项目展示 — 20+ 页面、40+ 组件
- GSAP ScrollTrigger 滚动动画、Cobe 3D 地球仪（about 页面）
- ISR 混合渲染 + Metadata API 动态 SEO

**日语学习平台（japanese-web）**：

- 教材 6 课完整内容（新标准日本語）、歌曲、柯南台词、旅行对话
- MDX 双编译路径 + 5-Tab 并行 UI（基础/应用/语法/词汇/练习）
- SakuraRain v4 物理樱花雨（60 片花瓣、3 层深度）
- 6 色主题系统（sakura/matcha/indigo/gold/warm/ink）

**开发者工具平台（tools-web）**：

- Terminal to HTML 转换
- SnippetBox 代码片段管理（@dnd-kit 拖拽排序 + 全文搜索 + localStorage 持久化）
- 歌词播放器（Ruby 假名注音）
- 世界时钟

**Spring Boot 后端（website-api）** — 36 个 API 端点：

- `/api/auth` — JWT 认证（HttpOnly Cookie + Token 黑名单 + 双维度限流）5 个端点
- `/api/blog` — 博客 CMS（CRUD + 标签 + 分类 + ISR 重验证）11 个端点
- `/api/comments` — 评论系统 5 个端点
- `/api/contact` — 联系表单 + 邮件通知 3 个端点
- `/api/users` — 用户管理 7 个端点
- 5 个数据模型（User, BlogPost, Comment, ContactMessage, Role）+ 22 个 DTO 定义

**共享库（packages）**：

- 5 个 npm 包（UI 组件库、自定义 Hooks、类型定义、工具函数、常量）
- 通过 Turborepo 构建缓存跨应用共享

**基础设施**：

- Monorepo 架构设计（pnpm + Turborepo）
- Docker 全栈开发环境（dev/prod 两套配置）
- Nginx 多域名反向代理（SSL + 限流 + CSP 安全头）
- 阿里云 ECS 部署 + Let's Encrypt 自动续期

### 6. 产品设计决策

**技术选型：**

- Next.js（非 Vite + React）：SSR/ISR 混合渲染 + Metadata API 简化 SEO + Standalone 输出减小 Docker 镜像
- Spring Boot（非 Node.js 后端）：企业级生态（Security + JPA + Mail + Async），Java 21 类型系统强
- pnpm + Turborepo（非 npm/yarn）：磁盘优化 + 构建缓存 + 跨包依赖清晰
- GSAP（非纯 CSS）：ScrollTrigger 滚动动画精度高、性能好

**数据流设计：**

```
用户请求 -> Nginx（SSL + 限流 + CSP）
  |-> /        -> website-web:3000（Next.js SSR/ISR）
  |-> /jp      -> japanese-web:3001（Next.js + MDX）
  |-> /tools   -> tools-web:3002（Next.js 纯前端）
  |-> /api/*   -> website-api:8080（Spring Boot）
       -> JwtAuthenticationFilter（Token 黑名单检查）
       -> Controller -> Service -> Repository（JPA）-> MySQL
```

**产品思维：**

- ErrorBoundary + 降级 mock 数据：网络故障不白屏
- HttpOnly Cookie + Token 黑名单 + 限流：防 XSS、防暴力破解
- ISR 1 小时重验证 + DTO 优化（500KB -> 15KB）：首屏快、流量省
- MDX 双编译路径 + 5-Tab UI：日语教材模块化，内容与代码分离
- SakuraRain v4 物理樱花雨：沉浸式日语学习体验
- 暗色模式 + 响应式 + Pangu 中英文间距：全球用户友好
- WCAG AA 对比度标准：无障碍访问
