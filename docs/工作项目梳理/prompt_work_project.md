# 工作 Project 对话 Prompt
# 用途：粘贴进 Claude App 工作相关的 Project 对话，提取简历素材

---

## 使用说明

打开你工作相关的 Claude Project 对话，把下面的内容直接粘贴进去，回车发送。
Claude 会根据这个 Project 里的所有历史对话来梳理信息。

---

## Prompt 正文

```
请根据我们在这个 Project 里所有的历史对话，帮我梳理简历素材。

我目前在找新工作，目标岗位是 AI Agent 开发工程师 / AI 应用开发工程师，有全栈开发背景，工作将近一年。

请按以下维度逐一梳理，每个维度都要有具体内容，不要写"暂无"或空白：

---

**1. 我实际负责了哪些开发工作**

不要照搬职位描述，要从我们的对话记录里提取我实际在做的事。
列出功能模块和具体任务，越细越好。
如果有多个项目，分项目列。

---

**2. 我是如何使用 AI 工具完成工作的**

从对话里找出具体例子：
- 我用 Claude Code / Claude / ChatGPT 解决了哪些具体的开发问题
- AI 工具在我的工作流里是什么角色（写代码、调试、设计方案、还是其他）
- 有没有我自己设计的 Prompt 策略或工作流
- AI 工具让我的效率提升体现在哪里（时间、质量、覆盖范围）

---

**3. 我遇到过哪些技术难题，怎么解决的**

从对话里找具体的技术问题和解决过程。
这些是简历上最有说服力的内容，越具体越好。

---

**4. 我的工作方式和思维方式**

从对话里推断：
- 我是怎么理解需求的（从业务出发还是从技术出发）
- 我在做技术决策时考虑什么因素
- 有没有我主动提出的方案或改进建议（而不只是执行任务）

---

**5. 从入职到现在，能力变化最明显的地方**

对比早期对话和近期对话，找出：
- 技术能力的具体变化（从不会到会，从粗糙到精细）
- 提问方式和思维方式的变化
- 独立解决问题的能力变化

---

**6. 最适合写进简历的 3-5 个亮点**

格式：亮点标题 + 一句话描述 + 支撑理由（从对话里找证据）

优先选择：
- 有 AI/LLM/Agent 相关的内容
- 有产品思维体现的内容
- 有可量化效果的内容

---

请直接输出结构化的内容，可以用的语言应该是简历语言（简洁、主动、可量化）。
如果某个维度在对话里找不到足够的信息，就直接说"对话记录中信息不足，建议补充：[具体建议]"。
```
