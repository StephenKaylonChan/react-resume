## 使用方法

### 更新简历内容
1. 修改对应的 Markdown 文件
2. 将更新后的内容同步到 React 组件中
   - `src/components/ResumeHeader.jsx` - 头部信息
   - `src/components/Resume.jsx` - 主要内容
   - `src/components/ResumeSidebar.jsx` - 侧边栏

### 版本切换
根据不同的求职目标，可以维护不同版本的内容：
- **全栈版本**: 强调前后端全栈能力
- **后端版本**: 侧重 Java 后端开发技能
- **前端版本**: 突出 React 前端开发经验

### 注意事项
- 更新内容时注意保持技术栈的一致性
- 确保渐变文字标签正确使用（如 `<GradientText gradient="gradient-react">React</GradientText>`）
- 日期格式统一为 `MM/YYYY` 格式
